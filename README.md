# Jeu-2D-Naruto-
Développement d’un jeu 2D interactif hors ligne, intégrant des éléments graphiques (images), des effets sonores et une musique de fond.
Le jeu inclut un système de score, la gestion des événements clavier et une logique de jeu simple offrant une expérience ludique et dynamique. 
Outils utilisés : Python, Pygame
